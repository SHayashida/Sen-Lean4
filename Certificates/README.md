# SAT/LRAT Certificate Verification

This directory is reserved for SAT/LRAT proof certificates used to verify finite
impossibility instances in Lean.

## BVDecide / LRAT workflow (recommended)

The module `SocialChoiceAtlas/Sen/BaseCase24/SATSenGenerated.lean` contains a
SAT encoding of the Sen (n=2, m=4) base case as a quantifier-free Bool/BitVec
formula intended for `Std.Tactic.BVDecide`.

To cache a solver run into an LRAT certificate and switch to fast checking:

1. Edit `SocialChoiceAtlas/Sen/BaseCase24/SATSenGenerated.lean` and replace
   `bv_decide` with `bv_decide?` in `sen24_unsat`.
2. Run Lean on the file via an *absolute path* so it can write the `.lrat` next
   to it:
   - `lake env lean /ABS/PATH/TO/SocialChoiceAtlas/Sen/BaseCase24/SATSenGenerated.lean`
3. Commit the generated `.lrat` file next to `SATSenGenerated.lean` and change
   the proof to:
   - `by bv_check "sen24_unsat.lrat"`
