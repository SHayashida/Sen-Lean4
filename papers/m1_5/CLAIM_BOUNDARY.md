# M1.5 Claim Boundary

This file is the paper-specific claim boundary for the M1.5 manuscript.

Current status:

- This workspace is a scaffold for the second paper.
- No paper-level empirical, theorem-level, or systems-performance claims are authorized by this file yet.
- Any future M1.5 claim must bind to:
  - explicit result directories,
  - committed or hashed artifacts,
  - a reproducible command path,
  - a commit hash or submission tag.

Before adding paper claims, record:

1. the exact result directories or manifests used,
2. the paper-facing figures/tables generated from those results,
3. the verification or audit commands that support the claim,
4. the submission tag intended to freeze the claim set.

Non-goal:

- This file does not change or broaden the claim boundary of the existing M1 manuscript under `paper/`.
