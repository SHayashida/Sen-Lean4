\begin{abstract}
This workspace is reserved for the second paper (M1.5) built on top of the shared SocialChoiceAtlas repository.
At the current stage, the manuscript records repository boundaries rather than paper claims:
shared code and data remain integrated on the main development line, while paper text, generated assets, and reproducibility notes are isolated under \code{papers/m1_5/}.
This scaffold is intentionally conservative so that M1 can continue under \code{paper/} without destabilization while M1.5 develops its own claim boundary and evidence trail.
