\section{Methods and Artifact Path}

The repository-level entrypoint for M1.5 paper assets is:

\begin{quote}
\code{python3 scripts/render_paper_assets.py --paper-id m1_5 --mode tiny}
\end{quote}

This entrypoint preserves the existing default behavior for M1 while allowing M1.5 to render into its own workspace.
When M1.5 matures into a claim-bearing manuscript, this section should be replaced with the actual experimental and verification pipeline, together with the result directories and hashes that back each paper-facing artifact.
