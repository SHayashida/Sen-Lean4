\section{Results Plan}

This scaffold does not yet assert any paper-level results.
Instead, it reserves a clean location for future M1.5 figures, tables, and result summaries.
When claims are added, they should reference explicit result directories, manifests, or proof objects rather than informal local state.

In particular, M1.5 should document:

\begin{itemize}[leftmargin=1.5em]
\item the exact atlas or experiment outputs used for each figure or table,
\item the command that regenerates each paper-facing asset,
\item the commit hash or submission tag that freezes the paper state.
\end{itemize}
