\section{Reproducibility Note}
\label{sec:repro}

This manuscript is grounded in a kernel-checked Lean module and an existing decision record rather than new solver experiments.
The main evidence sources used in the text are:

\begin{itemize}[leftmargin=1.5em]
\item \code{SocialChoiceAtlas/Sen/Lifting/ImpossibilityLift.lean} --- the M2 positive flagship theorem \(\mathtt{sen\_impossibility\_lifts}\) and its supporting helpers.
\item \code{SocialChoiceAtlas.lean} --- the root import line that wires the lifting module into the default \(\mathtt{lake\ build}\) (Decision 3 in the aim sign-off).
\item \code{SocialChoiceAtlas/Sen/BaseCase24/Theorem.lean} --- the read-only source of the ported \(\mathtt{sen\_not\_acyclic\_01}\) case-split that the lift re-states polymorphically.
\item \code{docs/m2\_scope\_wall.md} --- the Negative \#1 scope wall.
\item \code{docs/gates/m2/aim\_signoff.md} --- the four-decision record, including the ratification of the M2/M2.1 split.
\item \code{docs/gates/m2/impossibility\_lift\_prompt.md} and \code{docs/gates/m2/impossibility\_lift\_stage1\_plan.md} --- the staged construction prompts that produced the lift module.
\item \code{docs/no\_cycle\_interpretation\_note.md} and \code{docs/axiom\_semantics\_scaling.md} --- the prior repository notes that establish why the short-cycle encoding does not transfer past \(m = 4\).
\item \code{scripts/check\_acyclicity\_short\_cycles.py} --- the finite check that audits the short-cycle encoding at the base case.
\item \code{scripts/ci\_m2\_smoke.sh} --- the M2 smoke gate that fixes the invariants on the lifting module, the root import, the theorem signature, and the scope-wall document.
\end{itemize}

The construction branch is \code{codex/m2-bridge} and the corresponding pull request is \code{SHayashida/Sen-Lean4\#6}.

\subsection{Build commands}
\label{sec:build-commands}

The narrow build command that exercises the M2 positive flagship is:
\begin{quote}
\code{lake build SocialChoiceAtlas.Sen.Lifting.ImpossibilityLift}
\end{quote}

The M2 smoke-gate that pins the M2-specific invariants is:
\begin{quote}
\code{./scripts/ci\_m2\_smoke.sh}
\end{quote}

The full default \(\mathtt{lake\ build}\) exercises the lifting module via the root import in \code{SocialChoiceAtlas.lean}.

The manuscript itself builds with:
\begin{quote}
\code{make -C papers/m2 pdf}
\end{quote}
