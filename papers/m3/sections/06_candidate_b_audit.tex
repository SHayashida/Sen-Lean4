\section{Candidate B: Artifact-Backed Instantiation}
\label{sec:candidate-b}

This section instantiates M3-B using existing artifacts; it is not a Lean
formalization of those artifacts.  The bundled artifact-defined contract is
\[
  I =
  \{\mathtt{asymm},\mathtt{un},\mathtt{minlib},\mathtt{no\_cycle4}\}.
\]
The \(\mathtt{no\_cycle3}\) lever is inactive in this contract scope.  The split
implementation refines the liberalism block as
\[
  \beta(\mathtt{minlib})
  =
  \{\mathtt{decisive\_voter0},\mathtt{decisive\_voter1}\}.
\]
Candidate B is therefore non-atomic and falls under M3-B rather than M3-A.

\paragraph{Audit provenance.}
The M3 Candidate B audit is artifact-only; no new solver run is part of this
paper audit.  The main audit record is
\code{docs/m3_candidate_b_group_soundness_audit_result.md}, and the main
structured comparison artifact is
\code{results/20260401/candidate_b_minlib_granularity/comparison.json}.
The fully active bundled case is
\code{bundled/case_11101}, and the fully active split case is
\code{split/case_111101}.  Throughout this instantiation,
\code{no_cycle3} is fixed inactive.  The audit therefore establishes M3-B
only inside the bundled artifact-defined contract declared above.

  \item[Residual faithfulness.]
    The 16 block-aligned rows in \code{comparison.json}, all filtered to
    \code{no_cycle3=off}.
  \item[RawRep completeness.]
    The five split singleton repairs are
    \[
      \begin{gathered}
      \{\mathtt{asymm}\},\ \{\mathtt{un}\},\
      \{\mathtt{decisive\_voter0}\},\\

\end{description}

\paragraph{What would invalidate the instantiation.}
Candidate B would no longer instantiate M3-B under this contract if either
fully active residual were not UNSAT; any block-aligned comparison failed; the
split raw-repair family were incomplete; a non-singleton or missing singleton
changed the RawRep completeness claim; any grouped raw repair failed to map to
a bundled SAT residual; \(\PsiDeletionMonotonicity\) failed on the bundled
16-row lattice; \code{no_cycle3} were silently treated as active; or the
artifact-defined contract were silently upgraded to semantic or Lean-level
status.

Given the complete artifact-level enumeration of the split raw repairs,
Proposition~\ref{prop:audit-collapse} turns the grouped SAT checks into
unrestricted \(\GroupSoundness\).  Residual faithfulness is supplied by the 16
block-aligned status comparisons.  Therefore:

\begin{proposition}[Candidate B instantiation]
Candidate B instantiates M3-B under the bundled-level artifact-defined
contract.
\end{proposition}

The grouped family is
\[
  \bigl\{
  \{\mathtt{asymm}\},
  \{\mathtt{un}\},
  \{\mathtt{minlib}\},
  \{\mathtt{no\_cycle4}\}
  \bigr\}.
\]

Candidate B is raw-noncanonical and contract-canonical at the same time.
This does not upgrade the artifact-defined contract to a semantic or
Lean-level contract.  That upgrade is a separate truth/certificate obligation.
