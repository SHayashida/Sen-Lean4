\subsection{What Is Not Reproduced Here}

No new solver run is part of this preprint draft. No CNF or LRAT artifacts are
regenerated here. No Level C semantic-to-CNF correctness proof is provided.
The appendix binds the manuscript to documented evidence and source files; it
does not claim to regenerate the finite-audit pipeline or to verify generated
artifacts end to end. In particular, this appendix should not be read as a
claim that Python clause generation, SAT assignment semantics, or CNF artifact
correctness has been proved. Those remain future correctness obligations under
the paper's Level C terminology.
